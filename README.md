## Quick Start
 **Step: 1** Install yo
```bash
npm install -g yo
```

**Step: 2** install generator-htmlapp

```bash
npm install -g generator-htmlapp
```

**Step: 3** Finally, initiate the generator:

```bash
yo htmlapp
```
